El proyecto esta compuesto de un backend y un frontend los cuales se inicializancon:

Backend --> npm run dev (dentro de la propia carpeta del backend)
Frontend--> ng serve (dentro de la propia carpeta de frontend)

Backend: Este esta formado por un controlador donde encontramos las diferentes funciones: obtener la lista, añadir una nueva unidad y eliminar una unidad.
Un index con las correspondientes rutas de la api 
Y finalmente el modelo con el esquema de la farmacovigilancia. 

El frontend dispone de 2 componentes principales: 
-El formulario de registro. 
-El componente de la lista (donde generamos las card de cada una de las unidades). 

Cada uno de ellos con sus respectivos templates (html) y hojas de stylo (css). Personalmente trabajo con la hoja de estilos global que me gusta más.
